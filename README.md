
# InterviewPrep AI 🎯

InterviewPrep AI is a modern, premium web application designed to help developers and job seekers construct the ultimate interview toolkit. Built on the **MERN Stack** (MongoDB, Express, React, Node) and supercharged by **Google Gemini 2.0 Flash**, it generates highly tailored practice environments based on specific roles and experience levels. 

### ✨ Key Features
- **Intelligent Interview Generation:** Automatically generates specialized questions and answers tailored to your selected job title, experience level, and critical tech stacks.
- **Deep AI Explanations:** Don't just memorize the answer—ask the AI to explain the core concepts behind any generated interview question in real-time.
- **AI Resume Reviewer:** Native PDF ingestion pipeline that evaluates your uploaded resume for ATS formatting, action verbs, and actionable improvements to help you land the interview.
- **Session Tracking:** Persist, pin, and organize complex Q&A sets in your personal Dashboard.
- **Premium UI/UX:** A stunning, fully responsive frontend featuring 3D mesh gradients, Framer Motion animations, and custom glassmorphism components built with Tailwind CSS.

### 🛠️ Tech Stack
- **Frontend:** React, Vite, Framer Motion, Tailwind CSS, Axios
- **Backend:** Node.js, Express.js, JWT Authentication, Multer
- **Database:** MongoDB Atlas
- **AI Engine:** `@google/genai` (Gemini 2.0 Flash Multimodal Native System)
